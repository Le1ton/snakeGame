# 贪吃蛇小游戏 typescript + webpack

## 开始
    前往dist目录下载打包后的文件直接打开index.html即可开始游戏；
    克隆main目录后，vscode中输入yarn start亦可打开

## 组件
    Food 负责随机生成食物
    Snake 生成贪吃蛇主体
    ScroePanel 记录分数和等级，并且可以设置不同等级中蛇的速度快慢
    GameControl 游戏控制器，负责游戏的初始化以及控制蛇的移动等操作
